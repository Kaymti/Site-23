Nosso Objetivo

O projeto ------ tem objetivo de fazer com que informacoes geograficas do Vale do Paraíba sejam facilmente acessiveis a populacao em quaisquer dispositivo disponivel.

Como o site funciona

O nosso site operaciona utilizando HTML como base, CSS para formatação de paginas, JavaScript para operacao de elementos como o menu mobile, e por fim o uso da API QuickChart, uma variação do Chart.Js capaz de renderizar imagens, para geração dos nossos gráficos. O site é otimizado como responsivo, logo voce é capaz de acessa-lo através de dispositivos de resolução menores como celulares, Tablets ou até desktops Widescreen.
